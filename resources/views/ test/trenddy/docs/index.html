Introduction

Trenddy - HTML5 Blog Template, Responsive is fully responsive minimal template specifically built for agencies, shop, store and personal website. It can be used to build a perfect website for any agency business, creative agencies, architecture, designer, photographers, personal etc. It’s minimal design with super simple code gives all flexibility to develop to build a great website. Also available in dark version mode.

Please feel free to get back to me in case if you are having any question or feedback.
